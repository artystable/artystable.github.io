Privacy Policy

Short And Sweet

For the time being, I aim to collect minimal annonymous user data for various personal reasons, but fundamentally, to improve the quality of this website. I reserve the right to make changes to this policy as I see fit, at which point the changes shall be reflected in this Privacy Policy page and/or other relevant communication channel.

I use an implementation of my own making, which is open source and fully detailed here to collect minimal annonymous user data. In short, an annonymous fingerprint is generated for each visitor that enters the website. The collected annonymous data is then sent to a Google Analytics end-point to create a minimal record. The request made to Google Analytics specifically enforces that the IP be annonymized and as such the record is completely annonymous.

Google provides further information with regard to its data protection practices at https://policies.google.com/technologies/partner-sites,  including options you can exercise to prevent such use of your data.

While the site is hosted through GitHub/GitHub Pages, there is a degree to which user data might be collected beyond my control. You can find more information about how GitHub's privacy policies work by visiting The GitHub Privacy Statement and specifically, the GitHub Pages section. I found this Stackoverflow's law division answer particularly insightful in this regard.
